About this project

This project was developed as a personal project of Martin Roberts and George Sainsbury, who are both currently working in Technology Applications Delivery Branch of TSD. It is intended to showcase the power of ABS.Stat.

Technical Details

SearchABS is a lightweight, RESTful web application which communicates RESTfully with ABS.Stat. We take your query and try to match it the data available in ABS.Stat and make some good "guesses" when trying to deliver an answer. The application is built using Java, hosted on JBOS and returns a JSON result which is displayed using jQuery in HTML.
