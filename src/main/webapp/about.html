About this project

searchABS.com has been developed to showcase the power of ABS.Stat. It is natural language searching tool which allows users to search for single data statistical items within the ABS.Stat platform. This means that it will only be able to provide reasonable responses to queries where the answer is a single statistic that can be found within ABS.Stat. This project was developed as a personal project of Martin Roberts and George Sainsbury, who are both currently working in Technology Applications Delivery Branch of TSD.

Technical Details

SearchABS is a lightweight, RESTful web application which communicates RESTfully with ABS.Stat. We take your query and try to match it the data available in ABS.Stat using pattern matching, synonym replacement and Bayesian probability. The application is built using Java, hosted on JBOS and returns a JSON result which is displayed using jQuery in HTML.
